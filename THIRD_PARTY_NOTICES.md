# Third-party notices

Tabento contains no bundled third-party JavaScript library or package-manager dependency.

The interface can request font stylesheets for DM Sans, Inter, IBM Plex Sans, JetBrains Mono, Lora, Geist, and Geist Mono from Google Fonts. Those fonts are not vendored in this release; their upstream licenses and Google Fonts terms apply when fetched. System-font fallbacks remain available.

Favicon previews may use Google’s public favicon endpoint. Google is not affiliated with Tabento. Browser names and marks belong to their owners. Tabento icons, screenshots, and interface artwork are historical project assets unless a file states otherwise.
